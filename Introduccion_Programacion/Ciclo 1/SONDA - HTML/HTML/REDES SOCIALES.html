Lista De Definicion Avanzada.

HTML
Lenguaje marcado de HiperTexto utilizado para crear paginas Web
CSS
Lenguaje de Hojas de estilo que se utila para dar estilo a una pagina Web
JavaScript
Lenguaje de programacion utilizado para crear interactividad y dinamismo en las paginas Web
Bootstrap
Framework de diseño que incluye CSS y JavaScript predefinidos para crear sitios Web con estilo uniforme y coherente
jQuery
Biblioteca de JavaScript que simplifica la Manipulacion de elementos de HTML, Eventos y Animaciones de las paginas Web
